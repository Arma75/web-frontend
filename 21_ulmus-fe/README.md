# 🌌 Project Ulmus (울머스)
> **"From Constellation to Construction: Navigating the Infinite Architecture"**
> 
> **울머스**는 방대한 코드의 우주 속에서 당신의 설계를 별자리로 시뮬레이션하고,
> 단 한 번의 클릭으로 **Spring Boot** 백엔드 아키텍처를 실체화하는 지능형 코드 제네레이터입니다.

---

## 🌠 The Genesis: Why the Universe?

### **"방대함 속에 길을 잃다"**
처음 스프링 부트(Spring Boot)를 마주했을 때의 그 막막함을 기억하시나요?  
하나의 서비스를 만들기 위해 쏟아지는 수많은 어노테이션, 설정 파일, 그리고 복잡한 의존성들. 프로젝트가 진행될수록 기하급수적으로 팽창하며 수만 줄로 늘어가는 코드들을 보며 저는 깨달았습니다.

> **"우리가 만드는 것은 단순한 프로그램이 아니라, 하나의 거대한 우주다."**

암흑 같은 코드의 바다에서 아키텍처라는 **별자리(Constellation)**를 찾아내고, 복잡하게 얽힌 관계를 **코스믹 링크(Cosmic Link)**로 정의하는 것.   
**Ulmus**는 바로 그 지점에서 탄생했습니다. 보이지 않던 구조를 시각적 에너지로 치환하여, 개발자가 설계의 중심(Core)을 잃지 않도록 항해 지도를 제공합니다.

---

## 🛰 Beyond Simple ERD: The Real Utility

울머스는 시중에 흔히 널린 **단순한 ERD 드로잉 도구가 아닙니다.** 설계도가 '그림'으로 끝나지 않고 실제 '동작하는 물리적 환경'으로 전이되는 **Digital Twin** 경험을 제공합니다.

1.  **Celestial Mapping (설계):** 릴레이션을 별(Node)처럼 배치하고 관계를 잇습니다. 이는 단순한 선이 아니라 데이터가 흐르는 혈관입니다.
2.  **Luminous Analysis (분석):** 관계의 밀도가 높은 핵심 릴레이션은 스스로 빛납니다. 어디가 시스템의 병목이 될지, 어디가 가장 중요한 도메인인지 시각적으로 즉시 판별하세요.
3.  **Automatic Construction (생성):** 당신이 그린 별자리를 분석하여 **MyBatis + Thymeleaf** 기반의 **Layered Architecture** 소스코드 전체를 정교하게 깎아냅니다.
4.  **AI Reality Seeding (데이터):** AI가 당신의 도메인을 깊이 이해하여, `data.sql`에 '진짜 같은' 데이터를 채워 넣습니다. 서버를 띄우는 순간, 당신의 우주는 이미 생동감 있게 움직이고 있을 것입니다.

## ✨ Key Cosmic Features

### 🌌 Constellation-to-Code Engine
* **Instant Project Scaffolding:** 클릭 한 번으로 `Controller-Service-Mapper-DTO-Entity`에 이르는 풀스택 번들을 내려받으세요.
* **Semantic Attribute Sync:** PK/FK, 데이터 타입, 제약 조건이 DB 스키마와 Java 코드 사이에서 1%의 오차도 없이 동기화됩니다.

### 🤖 AI-Powered Cosmic Seeding
* **Context-Aware Intelligence:** AI는 단순히 무작위 문자열을 던지지 않습니다. `User` 테이블에는 실제 이메일을, `Order` 테이블에는 유효한 주문번호와 날짜를 생성합니다.
* **Domain-Specific Dummy:** 당신이 정의한 테이블 명과 코멘트를 기반으로 가장 '그럴듯한' 세계관을 SQL로 구축합니다.

### 🎮 Immersive Architect Cockpit
* **Resizable 3-Way Layout:** * **Left Wing:** 프로젝트 기술 스택 및 전역 빌드 옵션 관제.
    * **Main Canvas:** 데이터 별자리를 드래그 앤 드롭으로 설계하는 심우주 작업대.
    * **Right Wing:** 릴레이션과 속성의 정밀 수치를 튜닝하는 사이드 패널.
* **Dynamic Feedback UI:** 유리 질감(Glassmorphism)과 라임 네온 컬러가 조화된 인터페이스로 개발자의 몰입도를 극대화합니다.

## 🛠 Tech Stack: The Engine

### **The Core (Back-end)**
* **Framework:** Java 17 / Spring Boot 3.x
* **Persistence:** MyBatis (Full control over SQL)
* **View Engine:** Thymeleaf (Standard server-side rendering)
* **Intelligence:** AI API Integration for Contextual Seeding

### **The Shell (Front-end)**
* **Library:** React 18
* **Visual Engine:** React Flow (Neural-like Interaction)
* **Design System:** Cosmic Neon & Glassmorphism
* **Icons:** Lucide-react (Minimalist & Sharp)
